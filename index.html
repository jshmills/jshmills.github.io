Vanilla JS Snake Game

If you have some time to kill, enjoy a snake game I made.

This game was built with SOLID OOP principles in mind. I made use of constructor functions to build my objects and bind this. The snake is animate on a canvas element by moving rectangles through an array; moving the last indexed value to the front to give the appearance of movement. Restarting the game is just a matter of instantiating the objects again. In the future, I hope to break up my function expressions in order to better adhere to the "single responsibility principle" of OOP
